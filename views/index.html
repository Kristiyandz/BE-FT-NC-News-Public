API Endpoints

GET /api

Display all available API endpoints

GET /api/topics

Returns a JSON Object with all available topics

GET /api/topics/:topic_id/articles

Returns a JSON object containing all articles for a given topic

GET /api/articles

Returns a JSON object with all articles

GET /api/articles/:article_id/comments

Returns all comments for a single article

POST /api/articles/:article_id/comments

Adds a new comment to an article. The route requires a JSON body with a comment key and value pair e.g: {"comment":"This is my new comment"}

PUT /api/articles/:article_id

Increments or decrements the vote of an article by one. The route requires a vote query of 'up' or 'down' e.g: http://ncnewsapp.herokuapp.com/api/articles/:article_id?vote=down

PUT /api/comments/:comment_id

Increments or decrements the vote of a comment by one. The route requires a vote query of 'up' or 'down' e.g: http://ncnewsapp.herokuapp.com/api/comments/:comment_id?vote=down

DELETE /api/comments/:comment_id

This endpoint will find and delete a comment based on it's ID.

GET /api/users/:username

Returns a JSON object with the profile data for the provided user.
